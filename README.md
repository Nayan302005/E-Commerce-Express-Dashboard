## 🚀 Project Overview
This project is an interactive **E-Commerce Sales Dashboard** built using **Microsoft Power BI** to analyze sales performance, revenue trends, customer order behavior, and cancellation insights.

The dashboard provides real-time business intelligence through clean visualizations and KPI metrics.

---

## 📌 Features
✅ Revenue Analysis  
✅ Average Order Value Tracking  
✅ Cancellation Rate Monitoring  
✅ Lost Revenue due to Cancellations  
✅ Total Orders Overview  
✅ Product-wise Revenue Analysis  
✅ Category-wise Revenue Distribution  
✅ State-wise Revenue Insights  
✅ Monthly Revenue Trend Analysis  
✅ Interactive Date Filtering  
✅ Product Category Selection Buttons  

---

## 🛠 Tools & Technologies Used
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization

---

## 📊 Dashboard Insights
- Laptop category generated the highest revenue.
- Maharashtra contributed the maximum sales revenue.
- Cancellation rate is significantly high and impacts profitability.
- Revenue remained relatively stable across months before dropping sharply.
- Premium electronic products contributed major sales.
